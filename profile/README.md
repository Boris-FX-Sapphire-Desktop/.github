## 🚀 At a Glance
Boris FX Sapphire for Windows is a plugin suite that hands editors and motion designers a professional-grade effects library inside the tools they already use. Rather than piecing looks together node by node, you can apply polished glows, flares, and lighting in a few clicks and refine them from there. The Boris FX Sapphire After Effects, Premiere Pro, and DaVinci Resolve plugin installs once and shares the same effects across hosts, keeping your finishing pipeline consistent from project to project.

## 📥 Install Guide
1. Click the **Download** button above to get the latest version of Boris FX Sapphire.
2. Run the installer and select the hosts — After Effects, Premiere Pro, or DaVinci Resolve — you want the plugins added to.
3. Relaunch your host and open the effects panel to find Boris FX Sapphire installed and ready.

## 🎯 Quick Start
1. Open a project and select the clip you want to enhance.
2. Apply a Boris FX Sapphire effect such as a glow, flare, or transition.
3. Use the preset browser to pick a starting point.
4. Tune the parameters until the look suits your footage.
5. Render or export to bake the effect into your final sequence.

## ❓ FAQ
**Is Boris FX Sapphire free?**
No — it is a paid plugin suite from Boris FX. The vendor provides a free trial for evaluation, so you can explore the effects before buying a license, but a free download of the complete licensed suite is not offered; the trial is the proper way to try it.

**Which versions of Windows are supported?**
Boris FX Sapphire supports current 64-bit Windows releases such as Windows 10 and Windows 11, provided your host application also runs on that version.

**Is my data secure?**
The plugins render locally within your host application, so your footage and project files stay on your own workstation while you edit.
